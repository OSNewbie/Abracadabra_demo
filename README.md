# Abracadabra：魔曰

[<img src="https://img.shields.io/badge/license-AIPL%201.1-yellow"/>](LICENSE.md)

**Abracadabra(魔曰)** 是一个文本即时加密工具，也可用于加密文件，你正在查阅其Chrome浏览器插件实现。

Abracadabra 是表演魔术 (施魔法) 时所念的咒语，**魔曰** 是本项目的中文别名。
